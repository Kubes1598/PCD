📋 New Universal Game Logic Rules

  • ⚖️ Fair Play: If one player reaches 11 candies, the other player gets a final chance
  • 🤝 Draw System: If both players reach 11 candies without picking poison, game ends in a draw
  • 🔄 Turn Management: Proper turn switching ensures both players get equal opportunities
  • 🎯 Universal Application: Same logic applies to all game modes (Offline, Online, P2P, Friends)
  • 💔 Poison Rule: Picking opponent's poison still results in immediate loss
