# Game Asset Generation Guide

This document defines the visual assets required for **Poisoned Candy Duel**. It serves as a prompt guide for AI image generation agents (Midjourney, DALL-E, etc.) to ensure a consistent, premium aesthetic across all custom PNG/SVG files.

---

## 🎨 Global Art Direction
**Keywords**: Glassmorphism, 3D Render, Vibrant, High Gloss, Kawaii-Premium, Isometric.
**Style Description**: All items should have a consistent "tactile" 3D look with soft subsurface scattering and vibrant highlights. They should look edible and high-value, like polished jewels or premium candy.

---

## 📜 Technical Requirements for the AI Agent
1.  **Format**: PNG for complex 3D renders; SVG for minimalist logos.
2.  **Resolution**: 1024x1024 minimum for PNGs.
3.  **Consistency**: Ensure the lighting comes from the top-left for ALL 3D assets to prevent visual clashing in the grid.
4.  **Shadows**: Use soft contact shadows only; avoid harsh drop shadows.
